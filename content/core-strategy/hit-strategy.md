# 爆款战略

> 💡 **核心目标**：追求影响力最大化
> 🎯 **方法**：抢占市场洼地 + 优化传播形式 + 触发情绪共鸣

---

## 三大支柱

### 支柱1：市场洼地 ⭐⭐⭐⭐⭐

**定义**：搜索<10条的早期市场

**为什么重要**：
- 早期市场 = 需求大 + 供给少
- 先发优势 = 你的内容是用户看到的第一个

**执行标准**：
```
□ 搜索结果<10条？（必须）
□ 能24-48h完成？（抢先机）
□ 用户评论有需求？（真实痛点）
```

---

### 支柱2：视频优先 ⭐⭐⭐⭐

**数据**：视频转发率是文字的3倍

**为什么重要**：
- 降低理解门槛（视觉>文字）
- 提升传播效率（易转发）

**执行标准**：
```
□ 抽象概念有视频/动图？
□ 关键步骤有演示？
□ 开头3秒抓眼球？
```

**形式优先级**：
```
视频 > 图文 > 纯文字
```

---

### 支柱3：情绪共鸣 ⭐⭐⭐⭐⭐

**定义**：触发用户的情绪反应，驱动转发

**三种情绪**（优先级递减）：
1. **解脱感**："终于解决了困扰我的问题！"
2. **惊喜点**："原来还能这样！"
3. **认同感**："说出了我的心声！"

**执行标准**：
```
□ 用户读完会有什么感觉？
□ 开头有痛点前置？
□ 有对比感（旧方法vs新方法）？
```

---

## 与小白战略的关系

**小白战略 = 爆款战略的基础**

```
爆款战略
    ↓ 建立在

具体内容
```

**冲突时的优先级**：
1. 小白战略（基础）
2. 爆款战略（增强）
